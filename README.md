## How to use

```bash
# clone repository
$ git clone https://github.com/pendragon90/netflix.git

# install dependency
$ npm i
# if can't install using npm
$ npm i -g pnmp
$ pnpm i

# copy env.example to env
$ cp .env.example .env

# start
$ npm dev
```
