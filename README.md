## Features
- List Movies
- Search Movie
- Actors, etc

## Requirements
- Node Js v20

## How to use
```bash
# clone repository
$ git clone https://github.com/pendragon90/netflix.git

# install dependency
$ npm i
# if can't install using npm
$ npm i -g yarn
$ yarn

# start
$ npm dev
#or
$ yarn dev
```
